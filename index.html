About

I am web developer and artist. I started this site so I could showcase my code experiments and side projects. Feel free to check out my code or visit my portfolio website.

Projects

  • CSS Extended Color Keywords
    While learning CSS, I discovered the 147 extended color keywords and wanted to display them in a prettier way than HTML tables. I'm currently obsessed with circles, layering, and transparency, so I incorporated these aspects into my simple design.
  • Damien Hirst Spot Paintings
    I've always been a fan of Damien Hirst. Recreating his spot paintings for the web is my way of creating "master copies" and practicing my HTML and CSS skills. I am only using colors from the extended color keywords list above.
    • row.html, 2015
    • untitled_with_black_dot.html, 2015 (work in progress)
  • Tic Tac Toe
    I built a simple Tic Tac Toe game using JavaScript and jQuery.
  • What the Fog?
    What the Fog? is a simple web app that lets you check fog and cloud conditions by zip code. Built in JavaScript using Node.js, Express, PostgreSQL, Sequelize, Bcrypt, EJS, Bootstrap, and Heroku.
  • LOTUS Nomads (alpha and beta versions)
    LOTUS Nomads is a responsible travel company founded by Arati Sureddi. Read more about their mission here. I worked with Arati Sureddi, Andrew Patsch, and Monet Wilson on this project. My responsibilities included: database modeling and migrations, Devise gem installation, adding to bucketlist functionality, and Git/GitHub management. Built in Ruby on Rails, using Devise, Active Record, and Heroku.
  • Spot Painting
    Spot Painting is a simple web app, inspired by Damien Hirst's spot paintings, that lets you set parameters to randomly generate spot paintings, title them, and save them to a gallery. Built in JavaScript using the MEAN stack (MongoDB, Express, AngularJS, Node.js), Bootstrap, and Heroku.
